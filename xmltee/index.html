Setup

Click to view setup information.

To use this tool, you need to change DMS settings for your test unit to:

DMS setting name Setting
reporting.server http://xmltee.rumble.net/post/
reporting.omniture.reportsuites telstrabptboxdev,telstrabpiptvdev
reporting.reportinglevel 3

Note: This tool will only work with the T-Box XML POST Omniture beacons. The beacons generated using the JavaScript API (Apps and Games) will require a different test mechanism.
